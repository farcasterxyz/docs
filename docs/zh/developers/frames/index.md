# Frames 简介

::: info Mini Apps 正在进化！

<!-- prettier-ignore -->
我们近期发布了全新的 [Mini Apps 标准](https://miniapps.farcaster.xyz/){target="_self"}，以提供更优质的站内动态体验。**Frames v1 已被弃用，仅支持至 2025 年 3 月底。** 我们强烈建议所有新项目使用 Mini Apps。
:::

Frames 是一种直接在 Farcaster 社交动态中运行交互式应用的构建方式。

它们可用于为网页应用创建丰富的站内动态体验：

- 来自 [Zora](https://zora.co/) 和 [Highlight](https://highlight.xyz/) 的 NFT 可在动态中直接铸造
- 来自 [Paragraph](https://paragraph.xyz/) 的新闻简报可内联阅读并直接在动态中订阅
- 来自 [Polymarket](https://polymarket.com/) 的市场预测允许用户在动态中下注

或构建独立的互动体验如投票和游戏：

- 通过 [Safaricaster](https://warpcast.com/mattkim/0x3d165bb8) 在纳米比亚沙漠水坑的直播动态中发现并铸造野生动物
- 使用 [Paybot](https://app.paycaster.co) 向其他用户发起链上支付
- 通过 [Eventcaster](https://warpcast.com/toadyhawk.eth/0xcb4aefe8) 预约活动
- 在这款简单却风靡的社交游戏 [Yoink](https://warpcast.com/horsefacts.eth/0x70019199) 中夺取旗帜
- 基于你的 Farcaster 个人资料生成 [Waifu NFT](https://warpcast.com/horsefacts.eth/0xbc7d33ca)

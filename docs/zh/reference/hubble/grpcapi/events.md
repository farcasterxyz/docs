# 事件 API

用于订阅来自 Farcaster Hub 的实时事件更新

## API

| 方法名   | 请求类型         | 响应类型      | 描述                       |
| -------- | ---------------- | ------------- | -------------------------- |
| 订阅     | SubscribeRequest | 流式 HubEvent | 当新事件发生时进行流式推送 |
| 获取事件 | EventRequest     | HubEvent      | 当新事件发生时进行流式推送 |

## 订阅请求

| 字段        | 类型          | 标签     | 描述                         |
| ----------- | ------------- | -------- | ---------------------------- |
| event_types | [事件类型](#) | 重复字段 | 需要订阅的事件类型           |
| from_id     | uint64        | 可选字段 | 开始流式推送的事件 ID 起始点 |

## 事件请求

| 字段 | 类型              | 标签 | 描述 |
| ---- | ----------------- | ---- | ---- |
| id   | [uint64](#uint64) |      |      |

# Frames v2 已更名为 Mini Apps

<!-- prettier-ignore -->
自 2025 年初起，Frames v2 已更名为 Mini Apps。所有 Frames 文档和资源均已迁移至 [Mini Apps](https://miniapps.farcaster.xyz/){target="_self"} 生态系统。

Frames v2 已被弃用，其支持将持续至 2025 年 3 月底。我们强烈建议所有新项目使用 Mini Apps。
